# strcmd

```
String commands

Usage:
  strcmd [flags]
  strcmd [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  dec         Decrypt string
  decf        Decrypt strings from a file
  enc         Encrypt string
  encf        Encrypt strings from a file
  help        Help about any command
  keygen      Generate hex encoded encrypt key

Flags:
  -h, --help      help for strcmd
  -v, --version   version for strcmd

Use "strcmd [command] --help" for more information about a command.
```
